# **2. Two Distinct Systems: Interior Compass vs. Contact Resonance**

## **2.1 The Interior Compass (MBTI Layer)**

**What it describes:** Internal cognitive preference—how the person *experiences themselves* processing, evaluating, and deciding.

**Geometric Sources:**
- **Moon Signature** — the internal evaluation function (Somatic Basement)
- **MC/IC Axis** — purpose orientation and inner foundation
- **Saturn Signature** — long-arc structure and meaning-making
- **Venus-Moon-Saturn Interplay** — relational core architecture

**MBTI applies here.** This is the layer where E/I, N/S, T/F, and J/P preferences are inferred.

---

## **2.2 The Contact Resonance (Interface Layer)**

**What it describes:** External interface behavior—how *others experience the person* on first contact and in ongoing interaction.

**Geometric Sources:**
- **Ascendant** — the mask, first impression, interface style
- **Mars Signature** — ignition tempo, initiative pattern
- **Mercury Tempo** — speech speed, cognitive pacing in dialogue
- **Sun Sign/House** — visible identity expression

**MBTI does not apply here.** This layer describes contact resonance, not cognitive preference.

---

## **2.3 Why the Distinction Matters**

A person with:
- **Contact Resonance:** Aries stellium + Aquarius ASC → fast, bright, decisive contact style
- **Interior Compass:** Leo Moon + Scorpio MC + Saturn Rx → slow, depth-first, meaning-weighted processing

...may *appear* E/T-like on contact but *experience themselves* as I/F-like internally.

**Contact Resonance can mimic MBTI tendencies without representing them.**

If Raven conflates these layers, she will infer type from presentation rather than from cognitive preference—a category error.
